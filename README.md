# Haemoglobin Computational Analysis

This repository serves as a central hub for the computational modelling of haemoglobin structural dynamics. The project utilises Python to investigate the biophysical properties and conformational states of this essential protein.

## Project Overview

The repository is organised into two primary sections that explore distinct aspects of haemoglobin functionality:

* Part 1 focuses on the structural analysis of haemoglobin to quantify conformational transitions between the deoxy and oxy states. This section details the geometric extraction of critical pocket parameters.
* Part 2 investigates the interaction between haemoglobin and the therapeutic agent Voxelotor, which is used in the treatment of sickle cell anaemia. This section encompasses the computational workflow for binding site identification, molecular visualisation, and the empirical evaluation of Gibbs free energy to determine binding affinity.

Together these sections provide a comprehensive approach to understanding protein behaviour through structural bioinformatics and computational modelling.
