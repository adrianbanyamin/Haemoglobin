# Haemoglobin Conformational Analysis

This project performs computational analysis on haemoglobin structural data to investigate conformational transitions between the deoxy and oxyhemoglobin states.

## Overview
This script parses PDB files to calculate and compare pocket widths in different haemoglobin states. It serves as a tool for mapping coordination spheres to understand structural changes related to oxygen binding.

## Usage
Run the analysis script using Python:
```bash
python3 analyse_pocket.py
```
